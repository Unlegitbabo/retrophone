
## 1. Werkzeugliste

### 1.1 Unbedingt notwendig

| Werkzeug | Zweck |
|------|------|
| Regelbarer Lötkolben | Elektronik löten |
| Lötzinn | Elektrische Verbindungen |
| Multimeter | Spannung / Durchgang messen |
| Seitenschneider | Kabel kürzen |
| Abisolierzange | Kabel vorbereiten |
| Schraubendreher-Set | Telefon öffnen |
| Pinzette | Feinarbeiten |

---

### 1.2 Sehr empfohlen

| Werkzeug | Zweck |
|------|------|
| Dritte Hand / Platinenhalter | entspanntes Löten |
| Heißklebepistole | mechanische Fixierung |
| Heißluftfön | Schrumpfschlauch |
| Mini-Bohrmaschine / Dremel | Anpassungen |
| Logic Analyzer (optional) | Wählscheibe debuggen |

---

### 1.3 Software-Werkzeuge

| Tool | Zweck |
|------|------|
| Linux Laptop | Entwicklung |
| Arduino IDE oder PlatformIO | ESP31 Programmierung |
| USB-Treiber (falls nötig) | Serielle Verbindung |
| Git (optional) | Versionierung |
